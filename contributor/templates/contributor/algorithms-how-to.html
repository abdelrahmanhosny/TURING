TURING Algorithms for Computation with Finite Dynamical Systems

  1. How-to Guide

TURING utilized AlgoRun packaging system to ensure reliable algorithm execution. AlgoRun offers you the following:

  • You are not going to change your code. Yay!
  • You are not going to learn a new programming language.
  • The underlying Docker technology is going to ensure that your algorithm will always work the same way it works on your machine. No broken libraries or missing dependencies.

To start-off, navigate to AlgoRun and:

  1. Read the How-to package your software section. Get inspired by reading the examples section.
  2. Package your algorithm
  3. Push your algorithm Docker image to Docker Hub. (see how?)
  4. Publish your algorithm here!
